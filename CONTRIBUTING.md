# Contributing

PRs are always welcome to help make Sola better!

Pull requests will get squashed down into a single commit following
[the Conventional Commit syntax](https://www.conventionalcommits.org/en/v1.0.0/).

Good ways to get started:

- Add examples in the ./examples dir
- Improve documentation by making it more clear and adding missing docs

Ensure `just ok` passes on your machine before opening a PR.

If you're interested in helping maintain this project, please send an email to
[brett@brettchalupa.com](mailto:brett@brettchalupa.com).
